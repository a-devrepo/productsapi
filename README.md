# productsapi

API REST em desenvolvimento utilizando Java 25 e Spring Boot.

## Tecnologias

* Java 25
* Spring Boot
* Maven
* Docker
* PostgreSQL
* PgAdmin

## Ambiente

O projeto utiliza Docker Compose para criação dos containers do banco de dados PostgreSQL e do PgAdmin.

## Objetivo

Projeto de estudos para prática de desenvolvimento backend com Java e Spring Boot.
